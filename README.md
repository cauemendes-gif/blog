# Meu Blog Pessoal

Um template moderno, limpo e totalmente responsivo para blogs, desenvolvido puramente com HTML5 e CSS3.

## 🚀 Funcionalidades

- **Layout Responsivo**: Funciona perfeitamente em computadores, tablets e celulares.
- **Estrutura Semântica**: Utiliza as tags corretas do HTML5 (`<header>`, `<main>`, `<article>`, `<aside>`, `<footer>`).
- **CSS Grid e Flexbox**: Posicionamento moderno sem a necessidade de frameworks pesados como Bootstrap.
- **Variáveis CSS**: Paleta de cores centralizada no `:root` para fácil customização.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação de conteúdo.
- **CSS3**: Estilização e design responsivo.

## 🎨 Como Customizar as Cores

Para alterar a identidade visual do blog, basta abrir o código e modificar os valores das variáveis dentro do bloco `:root` no CSS:

```css
:root {
    --bg-body: #f4f7f6;      /* Cor de fundo da página */
    --primary: #008b8b;      /* Cor principal (botões, links, topo) */
    --text-dark: #2d3748;    /* Cor do texto principal */
    --white: #ffffff;        /* Cor dos cards */
    --gray-light: #e2e8f0;   /* Cor das bordas e divisores */
}
```
